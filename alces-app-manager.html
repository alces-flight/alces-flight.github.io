Alces Flight Application Manager Appliance

The Alces Flight Application Manager appliance allows you to store your favourite applications in one central repository, and serve them to one or many Alces compute environments. The Application Manager appliance allows you to easily install over 750 applications, compilers and accelerated libraries.

  • Learn More

The Alces Flight Application Manager appliance allows users to easily manage their applications, libraries and compilers from a single location. The Application Manager can be deployed one or many times to serve one or many environments.

The Application Manager appliance includes several key features, including:

Applications on-demand

The Application Manager appliance includes the Alces Gridware utility, simplifying installation and management of applications, libraries and compilers. With the Application Manager appliance deployed, applications can be installed onto the Application Manager appliance - making them available to other compute environments instantly.

Share applications among all of your environments

The Application Manager can be deployed once, to share applications among many compute environments - reducing overall configuration time of each new compute environment you deploy.

Provide applications for multiple OS

The Application Manager supports applications, compilers and accelerated libraries for Alces compute environments running both CentOS 6 and CentOS 7 compute hosts - simply install the Gridware package from your compute environment and the Application Manager will automatically sort the location for you.

Why should I run the Application Manager?

No other Cloud or baremetal computing platform provides applications like Alces - the Alces Gridware utility allows users of any skill level to install even the most complicated applications with one simple command.
